# Feature Freeze v1

FinFit v1 focuses on reducing complexity and proving an operating MVP.

## Included

- Latest youth benefits managed in an update-friendly structure
- Personalized first-pass benefit recommendation
- Easy explanations for difficult finance and policy terms
- API/cache/fallback operating evidence
- Automated tests

## Not Included

- Financial product enrollment
- Legal eligibility judgment
- Investment advice
- Replacement for banks or financial institutions
- Community, ranking, friend, or SNS features
- AI chatbot expansion
- Flutter migration
- Database replacement

## Reason

The largest current risk is not a lack of ideas. It is duplicated home screens, mixed routing, and hardcoded policy logic. v1 freezes feature scope so cleanup can finish.
